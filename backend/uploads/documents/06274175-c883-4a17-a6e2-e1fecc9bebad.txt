28. setMessages((prev) => [...prev, userMessage]); 의미?
이전 상태(state)를 기반으로 새로운 메시지를 추가하는 코드.
prev : 현재(이전)의 message 값.(가장 최신 상태)ㄴ
...prev : 전개 연산자(Spread Operator). 배열을 하나씩 펼친다.
message 수정 시 push를 사용하지 않는다. 상태(state)를 직접 변경하지 않고 새로운 배열을 만들어야
변경을 제대로 감지한다.

29. <Box sx={{}> 이건 무슨 방식?
sx : 스타일(css)을 작성하는 MUI 전용 문법이다.
첫번째 괄호({} : "JavaScript를 쓰겠다."
두번째 괄호({{}}) : JavaScript의 객체(Object)이다.

30. Hook?
- React 함수형 컴포넌트에서 상태(state), 생명주기, 라우팅 등 React의 기능을 사용할 수 있게 해주는 함수이다.
- 상태를 감지해서 화면을 재렌더링해준다.
- useXXX 형태로 시작한다.
- useState() : 상태 저장
- useEffect() : 컴포넌트가 실행될 때 API 호출 등에 사용
- useRef() : DOM을 참조하거나 값을 저장
- useParams() : URL 읽기
- useLocation() 현재 URL 정보 읽기

34. Pydantic?
Pydantic은 FastAPI에서 요청/응답 데이터를 검증하고 직렬화하기 위한 라이브러이다. 
BaseModel을 상속하여 데이터 모델 정의하고, 타입 검증을 자동으로 수행해 잘못된 요청을 방지한다.
또한 SQLAlchemy의 ORM 객체를 Pydantic 모델로 변환하여 응답을 생성할 떄도 사용한다.
